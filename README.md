# Fiscal SBC

Este projeto visa demonstrar conhecimentos sobre Game Development usando o Godot, e sobre o Código de Ética e Conduta Profissional da Sociedade Brasileira de Computação (SBC) no desenvolvimento de um jogo educativo ensinando os códigos de ética na computação.
Foi feito para a disciplina Informática e Sociedade na Universidade de Brasília.

<p>Como um avaliador técnico de um <strong>Órgão Fiscalizador de tecnologia</strong>, o seu trabalho é analisar uma série de projetos de software, submetidos por diversos clientes que trabalham em empresas de TI.&nbsp;</p>
<p>Seu objetivo&nbsp;é checar e decidir para cada projeto se ele segue ou viola o código de ética da SBC, de acordo com as diretrizes.</p>
<p>O jogador precisa analisar os casos de cada cliente da empresa de T.I e carimbar o arquivo do cliente como <strong style="color:#00c812">APROVADO</strong> ou <strong style="color:#f00f0f">NEGADO</strong>, analisando se o código de ética do caso foi violado ou não.&nbsp;</p>
<p>O jogador irá possuir um documento do código de conduta da SBC para consultar durante a análise.&nbsp;</p>
<p>O jogador precisa analisar o máximo de casos possíveis até o tempo limite acabar. No final de cada rodada é revelado quais casos foram avaliados corretamente ou incorretamente.</p>
